Projects

Designing Web Interface

2025 Web Interface

Project Summary

This visual project was developed as part of a UI/UX design practice using Figma. The goal was to create a clean and engaging user interface that showcases different nature destinations. The layout was structured to prioritize clarity, visual hierarchy, and responsive adaptability.

Objectives: Practice UI composition and layout with a strong visual hierarchy. Apply accessible design principles (color contrast, spacing, typography). Integrate high-quality visuals with supporting text. Simulate a realistic use case for a digital product interface.


About Design Decisions


The color palette reflects a natural theme, using greens and earthy tones to evoke a connection with nature.

Typography choices prioritize readability and balance.

A structured grid layout ensures clarity and consistency across content blocks.

Images are used to visually represent each nature location, paired with informative text to guide the user.

The overall interface maintains a soft, clean, and inviting visual style.
